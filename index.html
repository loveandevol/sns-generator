도움말

사용 방법
  1. 기본 정보를 입력해요.
  2. 댓글 또는 답글을 추가해요.
  3. 실시간으로 미리보기와 코드가 불러와져요!
TIPS
  1. 이미지 추가의 경우 코드로 이용할 때는 URL을, 이미지 저장으로 이용할 때는 업로드를 추천해요.
  2. 댓글 또는 답글은 추가 버튼을 누른 순서대로 입력돼요.
  3. 댓글과 답글은 수정이 불가능하며 전체 삭제 후 처음부터 다시 입력해야 해요.
  4. 이미지 편집 기능으로 원하는 부분만 크롭하고 크기를 조정할 수 있어요.
티스토리에 적용하는 법
  1. 글쓰기>오른쪽 상단 더보기 클릭>HTML 블럭
  2. 복사한 코드 붙여넣기
  3. 폰트 불러오기
    1. 완도청정바다체 다운로드(
    2. 스킨편집→html 편집→파일 업로드
    3. 다운로드한 서체 업로드
    4. CSS 최상단에 아래 코드 삽입
      @font-face {
    font-family: 'WandocleanseaB';
    src: url('images/WandocleanseaB.ttf') ;
    font-weight: normal;
    font-style: normal;
}
    5. HTML의 </head>안 위에 아래 코드 삽입
      <link rel="stylesheet" as="style" crossorigin href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable-jp-dynamic-subset.min.css" />
