SNS 생성기

사용 방법
  1. 기본 정보를 입력해요.
  2. 댓글 또는 답글을 추가해요.
  3. 코드 생성을 클릭 시 미리보기와 코드가 불러와져요!
TIPS
  1. 이미지 추가의 경우 코드로 이용할 때는 URL을, 이미지 저장으로 이용할 때는 업로드를 추천해요.
  2. 기본 정보는 수정 후 코드 생성을 재클릭하면 변경이 가능해요!
  3. 댓글 또는 답글은 추가 버튼을 누른 순서대로 입력돼요.
  4. 댓글과 답글은 수정, 삭제가 불가능하며 새로 고침 후 처음부터 다시 입력해야 해요.
